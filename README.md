# SupaChat

SupaChat is a conversational analytics application built with:

- Frontend: Next.js
- Backend: FastAPI
- Charts: Recharts

It allows users to query analytics data in natural language and view results as:

- chatbot responses
- SQL preview
- tables
- charts

## Supported Queries

1. Show top trending topics in last 30 days
2. Compare article engagement by topic
3. Plot daily views trend for AI articles
4. Show top 5 articles by views
5. Compare likes by topic
